1. Variáveis

Variáveis são contentores para armazenar valores de dados. Em Python, não precisas de declarar o tipo; basta atribuir um valor a um nome. 


# Variáveis são usadas para guardar dados (rótulos).
# O Python atribui o tipo de dado automaticamente.

nome_da_missao = "Astro Pi"
ano = 2025
velocidade_media_teste = 7.67
simulacao_ativa = True

# Podes imprimir variáveis usando f-strings (strings formatadas)
# Coloca um 'f' antes das aspas e as variáveis dentro de {}
print(f"Missão: {nome_da_missao}")
print(f"Ano: {ano}")
print(f"Velocidade de Teste: {velocidade_media_teste} km/s")

2. Tipos de Dados (Types)

Python tem vários tipos de dados incorporados. Estes são os mais importantes para nós. 


# str (String / Texto)
planeta = "Terra"

# int (Integer / Número Inteiro)
altitude_km = 408

# float (Número com casas decimais)
massa_kg = 420000.5

# bool (Boolean / Verdadeiro ou Falso)
em_orbita = True

# list (Lista - mutável, ordenada, permite duplicados)
tripulacao = ["Arnaldo", "Bruno", "Carla"]
dados_velocidade = [7.66, 7.67, 7.65, 7.67]

# dict (Dicionário - pares chave/valor, mutável)
# Útil para organizar dados
dados_da_foto = {
    "ficheiro": "imagem_01.jpg",
    "altitude": 408.5,
    "lat": 51.5072,
    "lon": -0.1276
}

# Podes aceder a dados numa lista pelo índice
print(f"Primeiro membro da tripulação: {tripulacao[0]}") # Output: Arnaldo

# Podes aceder a dados num dicionário pela chave
print(f"Ficheiro da foto: {dados_da_foto['ficheiro']}") # Output: imagem_01.jpg

# Podes verificar o tipo com a função type()
print(f"O tipo de 'altitude_km' é: {type(altitude_km)}")
print(f"O tipo de 'tripulacao' é: {type(tripulacao)}")
print(f"O tipo de 'dados_da_foto' é: {type(dados_da_foto)}")

3. Funções (Functions)

Funções são blocos de código reutilizáveis que só são executados quando chamados. Usamos def para as definir. Elas ajudam a organizar o código. 


# 1. Função simples (sem argumentos)
def saudar_equipa():
    print("Olá, equipa Astro Pi! Vamos programar.")

# Temos de "chamar" a função para a executar
saudar_equipa()

# 2. Função com argumentos (parâmetros)
def calcular_km_para_metros(kms):
    """Uma docstring explica o que a função faz."""
    metros = kms * 1000
    print(f"{kms} km são {metros} metros.")

# Passa o valor 408 para o argumento 'kms'
calcular_km_para_metros(408)

# 3. Função que "devolve" (return) um valor
# Esta é a mais importante para o nosso projeto!
def formatar_para_resultado(velocidade):
    """Formata a velocidade para o ficheiro result.txt."""
    # Formata para 4 casas decimais (ex: 7.1235)
    valor_formatado = f"{velocidade:.4f}"
    return valor_formatado

# O valor devolvido é guardado numa variável
velocidade_calculada = 7.6712345
velocidade_final = formatar_para_resultado(velocidade_calculada)

print(f"Valor a escrever no ficheiro: {velocidade_final}")

4. Loops (Ciclos)

Loops servem para repetir código. Os mais comuns são for (para iterar sobre uma sequência) e while (enquanto uma condição for verdadeira). 


# --- Loop 'for' (para iterar sobre uma lista) ---
tripulacao = ["Arnaldo", "Bruno", "Carla"]

print("Membros da Tripulação:")
for membro in tripulacao:
    print(f"- {membro}")

# Também podes usar range() para contar
print("Contagem decrescente:")
for i in range(3, 0, -1): # De 3 até 1 (não inclui o 0)
    print(i)
print("Lançar!")

# --- Loop 'while' (enquanto uma condição for verdadeira) ---
# (Isto é CRUCIAL para o vosso loop de 10 minutos!)

import datetime
import time # Para podermos fazer 'sleep'

# Define o tempo de início
tempo_inicio = datetime.datetime.now()
# Define um tempo limite curto para o exemplo (5 seg)
# Na missão, serão 10 minutos (timedelta(minutes=10))
tempo_limite = datetime.timedelta(seconds=5)

print(f"Iniciando simulação (loop 'while' de {tempo_limite.seconds} segundos)...")

while (datetime.datetime.now() - tempo_inicio) < tempo_limite:
    # O vosso código de recolha de dados (câmara, GPS) viria aqui
    print("...a recolher dados...")
    time.sleep(1) # Espera 1 segundo
    
print("Loop 'while' terminado! Limite de tempo atingido.")
